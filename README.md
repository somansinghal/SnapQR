# 🌟 About SnapQR

**SnapQR** is a powerful **QR Code + PDF Toolkit** that allows you to generate beautiful QR codes and manage PDFs directly in your browser.

Everything runs **locally in the browser** using JavaScript.

No backend required.

---

# ✨ Features

## 🔳 QR Generator

Create modern QR codes instantly.

✔ Gradient QR codes  
✔ Logo embedding  
✔ Multiple QR styles  

Supported styles

- Dots
- Square
- Rounded
- Classy

✔ Download QR as PNG  

---

## 📦 QR Templates

Generate QR codes instantly for:

📶 WiFi  
💬 WhatsApp  
🎥 YouTube  
📍 Google Maps  
💰 UPI Payment  

---

## 📄 PDF Tools

Powerful browser-based PDF utilities.

✔ Compress PDF  
✔ Merge PDFs  
✔ Split PDFs  
✔ Target file size compression  

---

## 📊 QR Analytics

Track QR usage.

✔ Scan counter  
✔ Device tracking  
✔ Heatmap visualization

# ⚙️ Installation

Clone repository

```
git clone https://github.com/somansinghal/SnapQR
```

Go inside project

```
cd SnapQR
```

Run locally

```
open index.html
```
